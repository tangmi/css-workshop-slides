## Extra stuff to fill time * **Child selector** (`div p`): Selects children any number of nested levels deep * **Immediate child selector** (`div ` > ` p`): Selects immedate children * **General sibling selector** (`h1 ~ p`): Selects all following siblings (same level) * **Immediate sibling selector** (`h1 + p`): Selects only the first following sibling * **Pseudo class selectors** (`:first-child`, `:first-letter`, etc): Special fun selectors whose names are self-explanatory * **Pseudo element selectors** (`:before`, `:after`): Inserts a pseudo "element" whose content is defined by the `content` property
