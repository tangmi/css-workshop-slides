Inline acts like regular text
and won't force following inline elements to wrap. Note that it doesn't really have a box model (doesn't honor margin or padding)
Block defaults to a 100% width
and forces all following elements to wrap. This is the default for all "block elements", such as div, all the headers, p, ol and ul, etc.
Table is stupid and you shouldn't use it
it acts like a block element, but entirely doesn't honor the width property if there's overflow.
